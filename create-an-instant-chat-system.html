Create an instant chat system

Launched in Jan 2020

Looking for a freelancer on a platform is like searching for products on an e-commerce site. First, they browse freelancers’ artwork (products). Second, when they see the one they like, they use an instant chat system to ask freelancers (sellers) for more information. Thus, the instant chat system seems to play an important role in this process. The following were the objectives of this project:

  • Allow users to instantly contact a freelancer.
  • Increase the revenue by attracting more freelancers to subscribe to Tasker premium.

Solution

1. Anyone, who has signed up for Tasker, can directly start a conversation with any freelancer on the website.

2. When freelancers receive a new message from clients, they must subscribe to Tasker premium to read the message. If freelancers haven’t subscribed, they will only be notified of the client’s needs summary and will not be allowed to see the client’s contact information.

Then, the developers and I created a diagram for the system:

Mockup

  • Display client’s needs and budget.
  • Users cannot see the message content since they haven’t subscribed to Tasker premium.
  • The call-to-action for subscribing.

After subscribing to Tasker premium, users can send a bid to the client and start a conversation with the client.
